## 校园点评平台

### 基础环境：

* Tomcat8.5
* Java 1.8
* MySQL 5.7

### 基本介绍：

这是初学servlet后做的一个小项目，项目很简陋，编程能力也欠缺。数据库文件在src/test.sql，创建数据库后更改src/main/java/dao/DBUtil.java中的`USERNAME`和`PASSWORD`属性即可。
